# Hospitality-Analytics-Dashboard
This Power BI dashboard analyzes hospitality business performance using booking and revenue data. The dashboard helps track occupancy, cancellations, revenue trends, and customer behavior.
Tools Used
Power BI
Key Insights
Revenue analysis
Occupancy percentage tracking
Booking cancellation trends
Weekday vs Weekend revenue comparison
Customer booking behavior
Features
Interactive slicers
KPI cards
Donut charts
Treemaps
Dynamic filters
Revenue trend analysis
Business Impact
The dashboard helps hospitality businesses understand customer trends, improve occupancy rates, and make better business decisions using data-driven insights.
